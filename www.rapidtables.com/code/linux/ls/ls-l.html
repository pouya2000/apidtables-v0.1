ls -l command in Linux

ls -l option flag lists with long listing format.

Syntax

$ ls -l [options] [file|dir]

Examples

ls, default list with short format:

$ ls
Desktop   Downloads Pictures Templates Videos
Documents Music     Public   todo.txt
$
