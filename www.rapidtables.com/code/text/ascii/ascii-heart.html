ASCII heart symbol

ASCII code for heart symbol (♥).

ASCII code does not include heart symbol.

ALT code

You can type it with ALT+3 with the numeric keypad: ♥

Unicode

Unicode is an extension of ASCII code and has 5 heart symbols:
